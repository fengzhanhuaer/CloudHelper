# CloudHelper 功能规划文档

CloudHelper 是一款旨在帮助开发者和运维人员高效管理云资源、服务器以及跨端同步的桌面级助手工具。

## 1. 核心定位
**“轻量、高效、安全的云端资源与运维管理桌面助手”**

## 2. 核心功能集 (MVP版本)

### 2.1 云主机与资产管理 (Cloud Resource Management)
- **多云平台接入**：支持配置 AWS、阿里云、腾讯云等主流云厂商的 API Key。
- **资源大盘**：在一个看板中查看名下所有云服务器 (ECS/EC2)、云数据库、对象存储的使用状态与到期提醒。

### 2.2 服务器连接与监控 (Server SSH & Monitoring)
- **快捷终端 (Quick SSH)**：内置轻量级 SSH 终端，支持密钥管理和一键连接。
- **性能状态监控**：通过云端服务或本地 Agent，实时展现服务器的 CPU、内存、网络 IO 等核心指标面板。

### 2.3 文件传输与云存储管理 (File & OSS Management)
- **可视化对象存储管理**：类似本地文件浏览器的体验，支持多云 OSS/S3 桶的文件的上传、下载、删除和预览。
- **跨端文件暂存区**：提供一个端到端的加密文件传输剪贴板，方便在本地与远程服务器之间快速流转脚本文档。

### 2.4 脚本与自动化任务 (Script & Automation)
- **代码片段/脚本库**：在本地保存常用的 Shell/Python 运维脚本。
- **一键下发执行**：选中目标服务器，一键推拉脚本并批量执行，收集执行日志。

## 3. 后续扩展规划 (v2.0+)
- **跨设备剪贴板同步**：支持端到端加密的跨设备文本复制粘贴。
- **Docker/K8s 面板**：支持直接管理远程服务器上的容器实例。
- **团队协作与权限分离**：后端服务支持多用户体系，实现团队视角的堡垒机功能。
